Message - This Java class represents a simple message system that stores message details such as a message index, recipient cell number, message text, and a unique 10-character message ID that is automatically generated using random letters and numbers. It includes methods to validate the message ID, check that the recipient’s phone number is correctly formatted (starting with +27), and ensure the message does not exceed 250 characters. The class also creates a message hash using parts of the message ID, message index, and the first and last words of the message to help uniquely identify it. In addition, a method is included to simulate different message actions such as sending, storing, or deleting a message based on user input. Overall, the class demonstrates key object-oriented programming concepts such as encapsulation, constructors, validation, and string manipulation.
MessageTest - This JUnit test class checks that the Messages class works correctly. It creates sample message objects using @BeforeEach and then tests different methods such as message length checking, recipient cell validation, message ID generation, message hash creation, and the sentMessage options. The tests make sure the program returns the correct results for both valid and invalid inputs. Overall, it helps confirm that all parts of the message system work as expected.
MainApp - This program is a simple console application that allows a user to register and log in. It collects a username, password, and phone number, then registers the user using the `login` class. After that, it asks the user to log in by entering their username and password, checks if the details are correct, and displays a message showing whether the login was successful or not.
